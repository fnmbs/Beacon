# OOU Campus Navigator

A modern campus navigation app designed to help students and visitors find locations and directions within Olabisi Onabanjo University (OOU).

## 🚀 Features

- Search for buildings and locations
- Interactive campus map
- Route directions within campus
- View departments and faculties
- Responsive design for mobile and desktop

## Tech Stack

**Frontend**

- React.js
- Tailwind

**Backend**

- Node.js
- Express.js

**Database**

- PostgreSQL

**Other**

- Axios

## Project Structure

Beacon/
├── client/ # React frontend
├── server/ # Node/Express backend
